JS

Javascript bir betik(script) dilidir. (java, server side bir programlama dilidir.) Derlenmeye ihtiyaç yoktur, tarayıcı tarafından yorumlanırlar. Düz html sayfalarına dinamiklik katmak için kullanılır.

Javascript kodlarının yazılması için

  • Elementin içine doğrudan yazılabilr.
  • Head bloklarının arasında script tagler içinde
  • sayfanın body bitişinden önce script tagleri içinde
  • external javascript dosyası içinde
Sayfanın açılış hızını yavaşlatmaması için body bloğunun bitişinden hemen önce yazılması önerilir.
